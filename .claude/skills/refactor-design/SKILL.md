---

プロジェクトルートにある `AGENTS.md` と `DESIGN.md` を読み込み、デザインルールを理解してください。

【タスク】
`src/` ディレクトリ配下のコンポーネントを分析し、`DESIGN.md` のルールに違反している箇所を洗い出してください。

【要件】
1. 以下の観点で違反箇所をすべてリストアップしてください:
   - カラー: `color-constants.ts` を経由せずにハードコードされている色（Tailwindクラスの直書き）
   - タイポグラフィ: フォントサイズ・ウェイト・`font-mono` の使い忘れ
   - スペーシング: 2・4・8の倍数ルールから外れている padding / margin / gap
   - ボーダーラジウス: `--radius` 系トークン以外の角丸指定
   - レスポンシブ: ブレークポイント `sm` の使い方の不統一
   - コンポーネント: Card・Button・Dialog・Badge・Table の使い方ガイドライン違反
2. 違反箇所ごとに「ファイル・行・修正内容」を明記してください。
3. 私が計画を承認した後にリファクタリングを実行し、`npm run lint` でエラーがないことを確認してください。
4. README.md に修正が必要な場合は更新してください。

まずは違反箇所のリストアップから始めてください。
